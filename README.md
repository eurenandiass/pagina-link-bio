Bio Page - Consultoria Imobiliária de Alto Padrão

Este projeto é uma central de links (Bio Page) desenvolvida com foco em branding pessoal e captura de leads para o mercado imobiliário do Rio de Janeiro. 

## 🚀 Tecnologias
- **HTML5 / CSS3**
- **Tailwind CSS** (Design minimalista e responsivo)
- **FontAwesome** (Iconografia profissional)

## 🎯 Diferenciais Técnicos
- **Layout "Magazine Style":** Hero section com foco total na imagem e tipografia refinada.
- **WhatsApp Tracking:** Links customizados com mensagens pré-definidas para identificar a origem do lead (Simulação, Consultoria ou Portfólio).
- **Mobile First:** Totalmente otimizado para navegação via smartphone.

## 🔗 Funcionalidades
- Catálogo de lançamentos integrado (Riva Incorporadora).
- CTA (Call to Action) direto para consultoria estratégica.
- Atalho para o sistema de simulação financeira.
